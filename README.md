Real Estate Agency Portal

Description of the Project
PHP and MySQL were used to create this project, which is a web-based real estate agency portal.

The platform permits: Agents to add and oversee properties listings

Renters and buyers can look through properties
Users can send inquiries about properties 
role-based login and authentication

Technolgies used for this project
The PHP, MySQL, phpMyAdmin, HTML, and XAMPP

Database Name: real_estate_portal_db

Features:
Registering and logging in as a user
Hashing and verifying passwords
Listings of properties
Details about the property
Submission of an inquiry
favorites page
Procedures that are stored
Views
Trigger
Management of sessions
Access control based on roles
